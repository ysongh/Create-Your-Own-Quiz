# Create Your Own Quiz

## IPFS
- Quiz are created and store on Fleek Storage. (https://ipfs.fleek.co/ipfs/ + HASH)
- Quiz images are store on Pinata. (https://gateway.pinata.cloud/ipfs/ + HASH)
